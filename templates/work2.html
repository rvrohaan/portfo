Web Scraping

This is a Python program I designed to get the top stories or articles for a website.

For this project I selected the website Hacker News. This is a website where you can find tonnes of articles regarding the trending and latest news. All these articles are rated by points. Since there are tonnes of articles I created this program to gather only the articles which have points greater than 100 to save time and assisting in reading only the good articles.
